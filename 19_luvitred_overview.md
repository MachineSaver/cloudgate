LuvitRED Overview
Source: https://cloudgateuniverse.com/docs/luvitred

Welcome!

LuvitRED is an easy-to-use graphical, "drag and drop & visual wiring" configuration environment for design and deployment of smart IoT Solutions on CloudGate. Before using LuvitRED please upgrade to the latest available Option CloudGate Firmware.
